# Trip Dashboard

Scaffolded React + Vite + TypeScript project with Tailwind CSS.

Quick start:

```bash
cd dashboard
npm install
npm run dev
```

This creates a basic dashboard with placeholder cards for:
- Itinerary
- Flights
- Weather
- Maps
- Budget
